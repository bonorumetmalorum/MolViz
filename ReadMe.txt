Follow the steps as described to compile and run the source code

1. Download and install Unreal Engine version 4.25 from the Epic Games Launcher which can be downloaded from the following link: https://www.epicgames.com/store/en-US/download?sessionInvalidated=true

2. After install Unreal Engine 4.25 clone this repo to your machine

3. open the project by double clicking on MolViz.uproject
